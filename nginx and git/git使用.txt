将github上的项目，克隆到本地一份
git clone git@github.com:账号名/项目名.git
------------------------------------------------



1、将文件添加到暂存区
git add 文件1，文件2，...

2、将暂存区的记录添加到仓库区
git commit -m '本次提交的说明信息'

3、将本地提交远程库
git push origin master

-------------------------------------------

1、回退历史版本到暂存区
git reset HEAD^或版本号


2、使用暂时区的内容恢复工作区的内容
git checkout -- 文件名


3、从远程库获取到本地
git pull
-------------------------------------------

查看暂存区的状态
git status
